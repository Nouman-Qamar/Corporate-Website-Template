# Corporate Website Template

A modern, fully responsive corporate website template for a digital technology solutions company — built with vanilla HTML, CSS, and JavaScript. Demo-branded as **Quantyra Technologies**, a fictional digital solutions company, to showcase a realistic end-to-end business site.

🔗 **Live Demo:** https://nouman-qamar.github.io/Corporate-Website-Template/

## 📱 Sections
1. **Navigation** – Sticky navbar with mobile toggle
2. **Hero** – Full-screen intro with call-to-action buttons
3. **About** – Company overview and key features
4. **Services** – Six service cards with icons
5. **Portfolio** – Project showcase with tags
6. **Testimonials** – Client reviews
7. **Contact** – Form with validation and contact info
8. **Footer** – Links and copyright

## ✨ JavaScript Features
- Mobile menu toggle
- Smooth scroll to sections
- Form validation with email regex
- Scroll-spying for active navigation
- Intersection Observer–powered scroll animations

## 🎨 Styling Highlights
- CSS custom properties for theming
- Gradient borders and backgrounds
- Responsive grid layouts
- Hover and focus states
- Custom animations and transitions

## 🛠️ Technologies
- **HTML5** – Semantic markup
- **CSS3** – Custom properties, Flexbox, Grid, animations
- **JavaScript** – DOM manipulation, event handling, form validation

## 📂 Project Structure
```
├── index.html   # Main HTML file
├── styles.css   # All CSS styles
└── script.js    # JavaScript functionality
```
